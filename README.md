## ⌨️ User Guide

1. Go to `backend` directory.
1. Go to `src/WebApi` directory.
2. Execute:

```bash
dotnet restore
dotnet build
dotnet
```


1. Go to `frontend` directory.
2. Execute: 

```bash
pnpm install
pnpm dev
```
